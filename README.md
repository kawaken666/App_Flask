# App_Flask
## アプリ概要
### 投稿記事閲覧アプリ
#### 実装機能
- 会員登録
- ログイン
- ログアウト
- 投稿記事一覧表示
- 投稿記事一覧の日付での絞り込み表示

#### 使い方
1. 以下herokuのURLにアクセス  
https://mighty-shore-18727.herokuapp.com/  
1. 最初に会員登録を行ってください  
    -未入力チェック以外のバリデーションは実装してないため、入力値はなんでも構いません
1. ログインしてください
1. 日付フォームで期間を指定して、日付指定ボタンを押下してください
1. ログアウトしてください
